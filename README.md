# 图库

使用git仓库作为图库，并引用到博客中。

## 从 jsdelivr 中国拉取图片

![](https://jsd.cdn.zzko.cn/gh/jackchoumine/jack-picture@master/webpack/webpack.png)

```bash
https://jsd.cdn.zzko.cn/gh/jackchoumine/jack-picture@master/webpack/webpack.png
```

```bash
https://jsd.cdn.zzko.cn/gh/{{owner}}/{{repo}}@{{branch}}/{{path}}
```
速度最快

> 在中国大陆不翻墙的速度
